// turbo-all
1. Stage all changes
   ```bash
   git add .
   ```

2. Commit changes with a clear message
   ```bash
   git commit -m "Update site content: contact form unification and branding font upgrade"
   ```

3. Push to the main branch
   ```bash
   git push origin main
   ```

4. Verify the deployment on GitHub Pages.
